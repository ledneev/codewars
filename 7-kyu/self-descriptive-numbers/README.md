<h2><a href=https://www.codewars.com/kata/56a628758f8d06b59800000f/train/javascript target="_blank">Self-Descriptive Numbers</a></h2><h3>7 kyu</h3><p><strong>A number is self-descriptive when the n'th digit describes the amount n appears in the number.</strong></p><p><strong>Example:</strong></p><p>For the number 21200:</p><ul><li>There are two 0's in the number, so the first digit is 2.</li><li>There is one 1 in the number, so the second digit is 1.</li><li>There are two 2's in the number, so the third digit is 2.</li><li>There are no 3's in the number, so the fourth digit is 0.</li><li>There are no 4's in the number, so the fifth digit is 0.</li></ul><p><strong>Numbers can be of any length up to 9 digits and are only full integers. For a given number, derive a function that returns:</strong></p><ul><li><code>True</code> if the number is self-descriptive</li><li><code>False</code> if the number is not.</li></ul>
